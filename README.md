#📌 Task Manager - Java

📖 Sobre o projeto

Sistema simples de gerenciamento de tarefas desenvolvido em Java, com foco em praticar lógica de programação e operações CRUD.

O sistema roda no terminal e permite:

Criar tarefas
Listar tarefas
Concluir tarefas
Remover tarefas
⚙️ Funcionalidades
✔ Adição de tarefas
✔ Listagem de tarefas
✔ Marcar tarefas como concluídas
✔ Remoção de tarefas
✔ Menu interativo via terminal
🛠 Tecnologias utilizadas
Java
ArrayList
Scanner
Programação Orientada a Objetos (básico)
Lógica de programação
💡 Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de praticar fundamentos da linguagem Java, manipulação de listas e construção de um sistema CRUD simples, simulando um mini gerenciador de tarefas.
